Scroll with Light Effect

A subtle moving light that follows scroll position — pure HTML/CSS with tiny JS to update variables.

Intro

This demo uses a fixed full-screen radial-gradient overlay. JavaScript updates CSS custom properties (--x and --y) while you scroll, repositioning the bright spot to give a "spotlight" effect that moves as the page scrolls.
